# Perfeita Luz - Lista de fotos sugeridas

Este documento lista as fotos recomendadas para o site e as resolucoes ideais para gerar via IA.
Use resolucao igual ou maior, mantendo a proporcao (aspect ratio).

Observacoes gerais
- Estilo editorial, premium, com foco em iluminacao.
- Paleta sugerida: grafite + dourado + cinza (seguir manual de marca).
- Preferir ambientes vazios ou com pessoas desfocadas, evitando destaque humano.
